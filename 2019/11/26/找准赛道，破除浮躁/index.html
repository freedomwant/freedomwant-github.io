找准赛道，破除浮躁

发表于 2019-11-26 更新于 2019-12-05 分类于 心理

起伏沉沦已然太久，回首这开学的接近三个月，自己收获甚微，那些立下的计划被悄无声息地放进了时间的碎片，我用漫无目的地琐碎来填充自己的大脑。有问题不可怕，问题越积越大才可怕，过去我就是这样，当然，确实是因为高三这个节点的我背负了太多的压力，无助。当然也是有利有弊，而当下自己如果依旧是一种过去的心态就太不应该了。

一直活在过去的恐惧当中，被恐惧吞噬不是一个理想青年期待的，一个没有原则的人，同样不值得获得赏赐。我擅长反思，但缺乏勇气，加上现在的一个抗干扰能力太差，除此无他。

为什么焦虑，为什么不安？原因很清楚，不确定性，以及周围朋友变相对比产生的差距感和失落感。接下来就是自己可以做的哪些改变呢？沉下心来，克制欲望，重拾理想，不被他人所裹挟。有自己的速度，自己的步伐，每天做出实际的成绩，激励自己，正能量一点。YES，I can do it.
